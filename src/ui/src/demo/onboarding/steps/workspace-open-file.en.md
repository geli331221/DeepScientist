Explorer is not only for browsing names. It is how you enter a concrete artifact.

- Click the highlighted file entry.
- The center surface will switch from the overview into the file itself.
- In this demo, the file is a markdown note. In real use, files like this act as your private notebook: a local-first Notion-style place to write plans, handoffs, findings, and working notes.
- This is how users move from “I know the structure” to “I am reading the actual project content.”
