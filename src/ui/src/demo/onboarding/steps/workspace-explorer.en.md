Explorer is the file-first view of the workspace.

This area answers a very practical question: *what durable files does this project actually have right now?* Whenever you want to verify that the graph and the narrative are backed by real outputs, this is the place to inspect.

- Open durable files here.
- Check whether the workspace already has useful artifacts like notes, run records, and result files.
- In real use, this panel is often how you decide whether the agent actually produced something reusable.
