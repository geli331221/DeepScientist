Click the highlighted **ArXiv** tab.

This opens the **ArXiv Library** for the current project.

- Papers you import yourself can appear here.
- Papers the agent reads and keeps for the project can also appear here.
- In this guided demo, `2509.26603` is preloaded as a visible example.

Treat this area as the literature shelf for the quest: titles, authors, categories, and reading progress stay attached to the workspace instead of disappearing into chat history.
